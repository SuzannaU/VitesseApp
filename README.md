# Vitesse
Vitesse is an app developed for the Vitesse Company. It is meant to simplify candidate management for Human Resources department.
It was built as part of the Android Developer course from OpenClassrooms.

## Functionalities
- Creation of a Candidate database (locally stored)
- Candidates can be created, modified, deleted
- Favorite functionality to select favorite ones
- Search widget to easily find candidates
- Currency conversion to display salary in GBP

## Tech stack
- Android Studio
- Gradle 9.1.0
- Kotlin 2.3.0
- Koin 4.2.0
- Room 2.8.4
- Retrofit 3.0.0
- Coil 3.3.0
- UI made with XML

## Testing
App is tested with Unit Tests (using Junit5 and Mockk)
